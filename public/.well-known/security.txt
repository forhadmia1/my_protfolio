# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in this portfolio website, please report it by emailing:

**Email:** forhadmia416@gmail.com

Please do not create a public GitHub issue for security vulnerabilities.

## Security Measures

This website implements the following security measures:

- HTTPS enforcement
- Content Security Policy (CSP)
- XSS protection
- CSRF protection
- Secure headers
